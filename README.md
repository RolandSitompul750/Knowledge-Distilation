# Knowledge Distilation – Model Development Repository

Repository ini merupakan proses dari pengembangan model dan eksperimen Temperature dan Alpha untuk distilasi. Tujuan utama dari pembuatan model ini adalah kompresi model dari model EfficientNetB5 sebagai model teacher dan model student sebagai model murid yang melalui proses Knowledge Distilation yang dimana pengetahuan dari model teacher diteruskan ke model student tetapi memiliki ukuran model yang lebih kecil.
